    "Brent": [30740, 28590, 30170, 30400, 30200],
    "Bromley": [18420, 17950, 19020, 19410, 19350],

Note

  1. It is mandatory to include 'area name'(case sensitive) in the csv file as the header and years in successive columns.
  2. If Importing JSON file make sure 'area name' as key and years as index and table_data as an object name
